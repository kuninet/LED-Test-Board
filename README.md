# KZ80-IOB LED Test Board

## 概要

![イメージ](image/LED-Testboard.JPEG)

* KZ80-IOBなどでLEDチカチカさせるためのテストボードです。
  * KZ80-IOB、K65-IOBなどのI/O端子(MIL 14Pコネクタ)へ接続して使用します。
* LED駆動のためにトランジスタアレイをつけてみました。
* ユニバーサル基板で試作する場合は、表面実装LEDで構成すると一列に並んできれいです。
